# Agent QC

Agent QC 定义 Lime 测试工作如何变成有计划、有证据、可批量、可评审的质量控制。

当 Agent 需要测试 Lime、创建 qcloop 批次、判断必跑门禁，或用证据汇报质量结论时，使用 Agent QC。
