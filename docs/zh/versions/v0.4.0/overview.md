# v0.4.0 概览

Agent QC v0.4.0 把项目案例研究扩展成更完整的跨 Agent 质量体系。它加入 Agent UI 风格的 runtime-backed surface 规则、Agent Skills 风格的编写实践、明确的证据契约、性能/可靠性指标，以及完整流程与分类参考。

## 亮点

- 明确标准定位是协议，不是单一产品检查表。
- 把 Agent UI surfaces 映射为 QC assertions：composer、message parts、runtime status、tool UI、HITL、task capsules、artifacts、timeline/evidence、sessions、team workbench。
- quickstart、gate matrix、classification、acceptance scenarios、verdict guidance 现在覆盖 CLI、TUI、WebUI、desktop GUI、browser automation、channel UI、scheduler、release 和 eval suites。
- evidence 与 waiver semantics 已作为可移植契约记录。
- performance/reliability metrics 成为 streams、tools、browsers、schedulers、release artifacts 的 QC 证据。
