# v0.1.0 overview

Agent QC v0.1.0 is the first product-focused draft for evidence-driven testing plans, gates, qcloop batches, verdicts, and reports.

- Scope: one initial product profile only.
- Primary integration: qcloop.
- Authority model: evidence-driven gates.
- Key gates: `verify:local`, `test:contracts`, `verify:gui-smoke`, Playwright, qcloop, and review.
