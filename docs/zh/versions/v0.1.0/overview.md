# v0.1.0 概览

Agent QC v0.1.0 is the first Lime-focused draft for evidence-driven testing plans, gates, qcloop batches, verdicts, and reports.

- Scope: Lime only.
- Primary integration: qcloop.
- Authority model: evidence-driven gates.
- Key gates: `verify:local`, `test:contracts`, `verify:gui-smoke`, Playwright, qcloop, and review.
