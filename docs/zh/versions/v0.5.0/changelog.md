# v0.5.0 变更记录

- 新增测试手段 taxonomy，覆盖 static/policy check、白盒单测、property/fuzz、golden transcript、snapshot、contract test、fake integration、黑盒 smoke、runtime E2E、surface E2E、replay、stress/chaos、安全对抗测试、semantic eval 和 release install smoke。
- 新增 runtime + UI evidence、TUI approval、provider adapter、browser agent safety、channel gateway、scheduler recovery、skill/plugin lifecycle 和 release confidence 的组合配方。
- 扩展最新规范、快速开始、门禁矩阵、项目分类、交互表面测试、最佳实践、来源索引、导航、README 和 LLM entrypoints，使其对齐 v0.5.0。
- 将通用标准页面改成产品无关协议表达；产品特定内容只保留在旧链接兼容页、版本历史或明确示例中。
- 将 package、public plan schema 和 public examples 升级到 `0.5.0`。
