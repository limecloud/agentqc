# v0.2.0 overview

Agent QC v0.2.0 generalizes the standard across Agent project types. Lime is now a profile and example, not the standard's scope boundary.

Key additions:

- project profiles;
- generic gate matrix;
- broader evidence and verdict model;
- case-study patterns from Codex, OpenClaw, Hermes Agent, and Lime.
