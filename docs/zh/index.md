# Agent QC

Agent QC 定义 Agent 项目如何分类、测试、留证、判定和报告。

适用对象包括运行时 CLI、SDK、工具网关、频道 Agent、GUI/TUI/桌面客户端、WebUI、浏览器自动化、Skills/插件、后台调度器、发行包和评测套件。
