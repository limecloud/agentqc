# v0.3.0 概览

Agent QC v0.3.0 把 v0.2.0 的分类骨架扩展为能指导真实 Agent 产品测试的实用标准。

关键新增：

- GUI/TUI/WebUI/desktop/browser/channel interaction surface 测试细则；
- Codex、Claude Code 本地快照、OpenClaw、Hermes Agent 和 Lime 的深度案例；
- Claude Code 风格 TUI/runtime 示例；
- 对 surface-specific `ui-interaction` gate 的更强证据要求。
